# lesbos-map

THE ISLAND — a live 2D-canvas map of Lesbos: one island that produced Sappho, Alcaeus, Terpander's lyre, Arion, Theophrastus, Aristotle's first biology (Kalloni lagoon), and an 18-Myr petrified forest. Click the sites, each links to its sphere. LESBOS. ROOT0, with AVAN. CC-BY-ND-4.0.

Self-contained interactive page (no external dependencies beyond fonts).
